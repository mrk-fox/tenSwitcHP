
# tenSwitcHP
**A ten inch rack compatible, partially 3D printed case for the HP ProCurve (2510G-24) switch(es).** <br>

![v1 assembly](https://github.com/mrk-fox/tenSwitcHP/blob/main/pictures/Screenshot%202026-03-23%20101011.png "v1 assembly")
*v1 assembly with "PCB" mounted*
<hr>
Hello and welcome to this little project! <br>
I had a 19 inch HP switch lying around and an idea in mind to build a 10 inch swich. The PCB of the 19 inch HP switch is slighly bigger than 10 inches and therefore requires a litte bit of  ✨angle✨ to fit in. In the next parts of this little doc you will be able to see the progress and version of this project as well as the current assembly.
Have fun!
